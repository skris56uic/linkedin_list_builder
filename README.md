# LinkedIn List Builder

A Chrome extension for extracting LinkedIn profiles and exporting them directly to a formatted CSV file.

## Features

- **Profile Extraction:** Automatically parse and gather names, headlines, location, and profile URLs directly from LinkedIn pages.
- **Two-Way Binding UI:** Select or deselect profiles directly from the page or using the extension's popup interface.
- **Smart Formatting:**
  - Cleans profile names, stripping out extraneous symbols, titles, and emojis.
  - Automatically splits full names into clean First and Last names.
- **CSV Export:** Allows exporting the selected profiles to a standard CSV file with clean data.
  - Generates ready-to-import rows with headers such as `First Name`, `Last Name`, `Title`, `Location`, `Company`, and `URL`.
  - Normalizes the Company URL by stripping out `http://`, `https://`, `www.`, and trailing slashes.

## How to Install (For Users)

1. **Build the extension:** Run `npm install` followed by `npm run build` to compile the TypeScript files and generate the final output.
2. **Open Chrome Extensions Page:** Navigate to `chrome://extensions/` in your browser.
3. **Enable Developer Mode:** Toggle the "Developer mode" switch in the top right corner.
4. **Load Unpacked:** Click the "Load unpacked" button and select the `build/` directory within this project.

## Development

This project uses TypeScript and compiles into the `build/` directory.

### Scripts

- `npm run build`: Compiles the TypeScript files into the target `build/` folder.
- `npm run lint`: Runs ESLint to check for code issues in the `src/` directory.
- `npm run lint:fix`: Automatically fixes any auto-fixable linting issues in the `src/` directory.

### Project Structure
- `src/`: Contains the TypeScript source code for the extension (popup, content scripts, background scripts, etc.).
- `build/`: The output directory generated by the build script (intended to be loaded into Chrome).
- `package.json`: Project metadata and build scripts.
